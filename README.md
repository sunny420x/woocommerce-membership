# ระบบ Membership สำหรับ WooCommerce
ซื้อสินค้าครบ 500 บาท รับ 1 points 

# เงื่อนไขภายในระบบ
- เมื่อสะสมจนครบ 10 points จะได้เลื่อนระดับเป็นลูกค้าระดับ Silver ได้ส่วนลด 1% ตลอดชีพ
- เมื่อสะสมจนครบ 20 points จะได้เลื่อนระดับเป็นลูกค้าระดับ Gold ได้ส่วนลด 3% ตลอดชีพ
- เมื่อสะสมจนครบ 30 points จะได้เลื่อนระดับเป็นลูกค้าระดับ Platinum ได้ส่วนลด 4% ตลอดชีพ
